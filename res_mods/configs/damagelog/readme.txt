
DamageLog - лог полученного, заблокированного и нанесенного урона..

			---   ---   ---	
 
Файлы модификации:

..\World_of_Tanks\res_mods\0.9.х\gui\scaleform\..
 - DamageLog.swf - flash файл, отображает графическую часть модификации..
 
..\World_of_Tanks\res_mods\0.9.х\scripts\client\gui\mods\..
 - mod_damagelog.pyc - скрипт файл, отвечает за логическую работу модификации..
 
..\World_of_Tanks\res_mods\configs\DamageLog\..
 - blocked.cfg - конфиг файл, отвечает за настройку блока с отображением заблокированного урона игроком..
 - damaged.cfg - конфиг файл, отвечает за настройку блока с отображением нанесенного урона игроком и его совзводными..
 - fire.cfg - config file is responsible for setting the unit to display a pop-up message on the fire ..
 - log.cfg - конфиг файл, отвечает за настройку блока с отображением лога полученного урона и попаданий игроком..
 - message.cfg - конфиг файл, отвечает за настройку блока с отображением всплывающих сообщений с полученным уроном и попаданий игроком..
 - timer.cfg - конфиг файл, отвечает за настройку блока с отображением обратного таймера с приблизительным временем перезарядки противника, попавшего по игроку..
 - sound.cfg - конфиг файл, отвечает за настройку звукового сопровождения событий..
 
0.9.х - текущая версия игры (например, 0.9.0)..
